make.ycor.matrix {slidaRtools}R Documentation

Make a Y-coordinate matrix

Description

Create a matrix that contains the Y-coordinate of each cell as cell value

Usage

make.ycor.matrix(minx = 0, maxx, miny = 0, maxy, res = 1)

Arguments

minx

minimal X-coordinate

maxx

maximal X-coordinate

miny

minimal Y-coordinate

maxy

maximal Y-coordinate

res

side length of one cell

Value

matrix of Y-coordinates

Author(s)

Nikolai Knapp, nikolai.knapp@ufz.de

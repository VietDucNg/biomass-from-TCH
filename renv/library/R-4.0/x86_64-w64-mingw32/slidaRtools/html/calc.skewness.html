Calculate Skewness

Description

Function for calculation of skewness of an empirical distribution

Usage

calc.skewness(vec)

Arguments

vec

Vector of data

Value

Skewness (negative = left-skewed / positive = right-skewed)

Nikolai Knapp

Examples

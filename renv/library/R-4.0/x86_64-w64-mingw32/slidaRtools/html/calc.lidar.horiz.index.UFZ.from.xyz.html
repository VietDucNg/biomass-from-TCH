calc.lidar.horiz.index.UFZ.from.xyz {slidaRtools}R Documentation

Calculate horizontal structure index from Lidar point cloud

Description

Function for horizontal index from UFZ (inspired by TUM index for field data) calculated from Lidar XYZ-table data (point cloud or CHM)

Usage

calc.lidar.horiz.index.UFZ.from.xyz(XYZtable)

Arguments

XYZtable

Lidar point cloud or CHM in XYZ-table format

Value

Index of horizontal forest structure

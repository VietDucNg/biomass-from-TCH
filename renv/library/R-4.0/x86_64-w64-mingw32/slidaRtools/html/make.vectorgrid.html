make.vectorgrid {slidaRtools}R Documentation

Make a vectorgrid

Description

Function that creates as vectorgrid of rectangular cells.

Usage

make.vectorgrid(
  ll.x = 0,
  ll.y = 0,
  size.x = 1,
  size.y = 1,
  n.x = 1,
  n.y = 1,
  rot.angle = 0
)

Arguments

ll.x

X-coordinate of the lower left corner

ll.y

Y-coordinate of the lower left corner

size.x

Side length of one cell in X-direction

size.y

Side length of one cell in Y-direction

n.x

Number of cells in X-direction

n.y

Number of cells in Y-direction

rot.angle

Rotation angle clockwise in degrees

Value

SpatialPolygonsDataFrame

Examples

testgrid.spdf <- make.vectorgrid(ll.x=0, ll.y=0, size.x=2, size.y=3, n.x=10, n.y=5)
plot(testgrid.spdf)
sub.spdf <- subset(testgrid, ID==1)
plot(sub.spdf, add=T, col="red")
sub.spdf <- subset(testgrid, ID==2)
plot(sub.spdf, add=T, col="green")
sub.spdf <- subset(testgrid, ID==11)
plot(sub.spdf, add=T, col="blue")

make.zcor.array {slidaRtools}R Documentation

Make a Z-coordinate array

Description

Create a 3D array that contains the Z-coordinate of each cell as cell value

Usage

make.zcor.array(minx = 0, maxx, miny = 0, maxy, minz = 0, maxz, res = 1)

Arguments

minx

minimal X-coordinate

maxx

maximal X-coordinate

miny

minimal Y-coordinate

maxy

maximal Y-coordinate

minz

minimal Z-coordinate

maxz

maximal Z-coordinate

res

side length of one cell

Value

3D array of Z-coordinates

Author(s)

Nikolai Knapp, nikolai.knapp@ufz.de

reproject.points {slidaRtools}R Documentation

Reproject points in space from one coordinate reference system (CRS) to another

Description

Function that reprojects X- and Y-coordinates between different CRS.

Usage

reproject.points(xcor, ycor, in.crs, out.crs)

Arguments

xcor

Vector of X-coordinates

ycor

Vector of Y-coordinates

in.crs

Original CRS of the data

out.crs

Target CRS to which the points should be projected

Value

SpatialPolygonsDataFrame

Examples

latlon <- CRS("+init=epsg:4326")
utm33n <- CRS("+init=epsg:32633")
reproject.points(xcor=c(0, 100), ycor=c(0, 100), in.crs=utm33n, out.crs=latlon)

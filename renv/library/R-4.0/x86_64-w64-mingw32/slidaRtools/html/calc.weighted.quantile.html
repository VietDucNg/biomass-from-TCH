calc.weighted.quantile {slidaRtools}R Documentation

Calculate weighted quantile

Description

Function for weighted quantile calculation in case of already aggregated data (e.g. Lidar profile)

Usage

calc.weighted.quantile(val, wt, quant)

Arguments

val

Vector of values

wt

Vector of weights (e.g. counts or relative proportions)

quant

Quantile as float between 0 and 1

Value

Quantile

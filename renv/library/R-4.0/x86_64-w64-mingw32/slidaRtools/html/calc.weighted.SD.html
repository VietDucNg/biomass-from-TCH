calc.weighted.SD {slidaRtools}R Documentation

Calculate weighted standard deviation

Description

Function for weighted SD calculation in case of already aggregated data (e.g. Lidar profile)

Usage

calc.weighted.SD(val, wt)

Arguments

val

Vector of values

wt

Vector of weights (e.g. counts or relative proportions)

Value

Standard deviation

Examples

val.vec <- c(3, 5, 4, 1)
wt.vec <- c(1, 1, 5, 5)
weighted.SD(val=val.vec, wt=wt.vec)

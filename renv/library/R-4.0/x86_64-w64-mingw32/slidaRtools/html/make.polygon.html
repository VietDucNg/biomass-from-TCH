make.polygon {slidaRtools}R Documentation

Make a polygon

Description

Function that creates a spatial polygon data frame from given vertex coordinates.

Usage

make.polygon(vertex.mx)

Arguments

vertex.mx

Matrix with two columns (X- and Y-coordinates) where each row represents one vertex

Value

SpatialPolygonsDataFrame

Examples

test.spdf <- make.polygon(vertex.mx=matrix(c(1,1,5,1,3,3), ncol=2, byrow=T))
plot(test.spdf)
